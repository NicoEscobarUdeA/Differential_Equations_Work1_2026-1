# Differential_Equations_Work1_2026-1


## File Descriptions

### 1. `Primera_EDO.py`
An introductory script that analyzes the differential equation:
`3y + 9y' = 7xy³`.

- **Functionality**:
  - Visualizes the slope field of the ODE.
  - Solves the ODE symbolically using `SymPy`.
  - Plots the family of solutions for different values of the integration constant `C`.

### 2. `AplicationAllModels.py`
The core Python simulation. Models a solar battery with the following characteristics:

- **Parameters**: Capacity `Q_max = 75 Ah`, power consumption `P_elec = 80 W`, initial SOC `50%`.
- **Solar power input**: `P_in(t) = max(0, 100 * sin(π * t / 24))` (positive only).
- **Voltage model**: `V = 22 + 4 * (Q / Q_max)`, varying between 22V and 26V.
- **Differential equation**: `dQ/dt = (P_in - P_elec) / V`, with limiters to prevent overcharging and complete discharge.
- **Analysis**:
  - Simulation of the nominal case for 48 hours.
  - Two extreme scenarios: overcharge (`P_elec = 40 W`) and overdischarge (`P_elec = 120 W`).
  - Comparison with a constant voltage model (`V = 24V`).
  - Exports all results to CSV files for subsequent analysis.

### 3. `Comparison.py`
Script that compares the results obtained with Python and OpenModelica.

- **Functionality**:
  - Loads CSV files generated by both environments.
  - Interpolates data for point-by-point comparison.
  - Generates overlay and difference plots to validate both models.
  - Produces a summary table (`results_table.csv`) with final SOC values for each scenario.

### 4. OpenModelica Files (`.mo`)
Models developed in Modelica for system simulation. Each represents a different scenario:

- **`VariableVoltage.mo`**: Main model with variable voltage, analogous to the Python version.
- **`ConstantVoltage.mo`**: Version with constant voltage (24V) for comparison.
- **`Overcharge.mo`**: Overcharge scenario with `P_elec = 40W`.
- **`OverDischarge.mo`**: Overdischarge scenario with `P_elec = 120W`.

### 5. Data Files (`.csv`)
Files generated by OpenModelica and Python containing simulation variables (time, Q, SOC, V, currents, etc.). These form the basis for analysis and comparison.
